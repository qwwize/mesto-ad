# Проект Mesto

Проектная работа курса "Веб-разработчик" от Яндекс.Практикум. Интерактивная страница для публикации фотографий с возможностью редактирования профиля, добавления и удаления карточек, постановки лайков.

## Технологии

- HTML5
- CSS3
- JavaScript (ES6+)
- Vite
- API

## Функциональность

- Редактирование профиля пользователя
- Обновление аватара
- Добавление новых карточек с фотографиями
- Удаление своих карточек
- Постановка и снятие лайков
- Просмотр изображений в полноразмерном режиме
- Модальное окно подтверждения удаления
- Статистика пользователей (при клике на логотип)
- Валидация форм
- Улучшенный UX (индикаторы загрузки)

## Установка и запуск

```bash
# Установка зависимостей
npm install

# Запуск dev-сервера
npm run dev

# Сборка проекта
npm run build
```

## Публикация проекта

Проект автоматически публикуется на GitHub Pages при каждом push в ветку `main` через GitHub Actions.

**Ссылка на опубликованный проект:** https://qwwize.github.io/mesto-production/

### Запуск деплоя вручную

Если нужно запустить деплой вручную без создания нового коммита:

1. Перейдите в раздел **Actions** приватного репозитория
2. Выберите workflow **Deploy to Public Pages Repository** в левом меню
3. Нажмите кнопку **Run workflow** справа
4. Выберите ветку и нажмите **Run workflow**
